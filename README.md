# turtleOS
**The official microcontroller navigation system for the hope turle**

---

## Overview

The hope turtle project envisions automomous marine vehicles made from 95% organic material, that are able to self-navigate within +- 5Km to a coastal destination to deliver food, building materials and... hope.  This repository holds the code development for the microcontroller operating system that enables the operation, power management, sensor integration and overall navigation of the turle.  Our codebase is in mycropython and is focused on the XIAO ESP32-S3.
